🦴 01 Section Cheat Sheet

1. Core Section Definitions (7 Section Types, 8 Dimensions incl. Section Order)

H (Hook) Attention CaptureInterrupt auto-scrolling, compete for cognitive bandwidth (Required)
Br (Bridge) Relevance BuildingEstablish context and personal relevance (Optional)
P (Problem) Tension BuildingExpose the gap between current state and ideal (Strongly recommended)
Re (Resolution) Tension ReleaseRelieve tension, show improvement path (Highly recommended)
Rv (Reversal) Cognitive RewriteOverturn existing beliefs, eliminate potential resistance
V (Value) Decision StabilizationCore value of product/service (Strongly recommended)
C (CTA) Action TriggerConvert preference into action (Required)

Tier A = 4 sections or fewer (short & fast); Tier B = 5 sections (standard persuasion); Tier C = 6+ sections (deep narrative)

5. Emotion Arc (Node Array)

Each Section corresponds to an emotion node, annotating the emotion label and intensity value (0-10) for that section. The full arc is composed of a node array nodes[].

FieldTypeDescription
sectionenum (H/Br/P/Re/Rv/V/C)Section it belongs to
emotionstringPrimary emotion label for the section (e.g. curiosity, anxiety, relief, desire, urgency)
intensitynumber 0-10Emotion intensity, 0 = calm, 10 = extreme

Example: [{section:"H", emotion:"curiosity", intensity:7}, {section:"P", emotion:"anxiety", intensity:8}, {section:"V", emotion:"desire", intensity:9}, {section:"C", emotion:"urgency", intensity:6}]
